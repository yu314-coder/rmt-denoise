📐 Cubic Equation & Method

Cubic Polynomial Equation:

(z·a)·s³ + [z·(a+1) + a·(1-y)]·s² + [z + (a+1) - y - y·β·(a-1)]·s + 1 = 0

Stieltjes Transform:

The roots s represent the Stieltjes transform of the eigenvalue distribution:

s(z) = ∫ (1/(λ - z)) dμ(λ)

Im(s) Interpretation:

• Im(s) relates to the density of eigenvalues

• Re(s) relates to the principal value

• Complex roots indicate overlapping eigenvalue support

• Real roots indicate separated eigenvalue bands

📐 Matrix Construction & Theory

Matrix Model:

B_n = S_n × T_n

Sample Covariance Matrix:

S_n = (1/n) × X × X^T

Where X is a (p × n) random matrix with Gaussian entries

Spike Matrix:

T_n = diag([a, a, ..., a, 1, 1, ..., 1])

• First β×p entries are a (spike values)

• Remaining (1-β)×p entries are 1

Parameters:

• y = p/n: Aspect ratio (calculated from image dimensions)

• a: Spike value (default: 2.0)

• β: Fraction of spiked eigenvalues

• n: Number of samples (images)

• p: Dimension (image pixels)
